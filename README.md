## Shipped Products

### [kr-bid](https://github.com/HyunjoJung/kr-bid)

조달청 입찰공고·낙찰 데이터와 HWP/PDF/XLSX 첨부문서를 수집·구조화해 검색, 자격조건 확인, 낙찰 분석과 MCP 연동을 제공하는 공공조달 서비스입니다.

- Live: [kr-bid.com](https://kr-bid.com)
- Stack: Rust, Axum, SQLx, PostgreSQL, Next.js, MCP

### [SafetyEye](https://github.com/HyunjoJung/safety-eye)

Android·iOS 기기를 온디바이스 AI 안전 카메라로 활용하고, 감지 기록과 공공데이터를 연결해 산업안전 보고서를 생성하는 서비스입니다.

- Live: [safetyeye.hyunjo.uk](https://safetyeye.hyunjo.uk)
- Stack: Kotlin, Swift, LiteRT-LM, Cloudflare Workers, MCP

### [SheetLink](https://sheetlink.hyunjo.uk)

파일을 서버에 저장하지 않고 엑셀 하이퍼링크를 추출·병합하는 웹 도구입니다.

- Stack: ASP.NET Core, Blazor, OpenXML
